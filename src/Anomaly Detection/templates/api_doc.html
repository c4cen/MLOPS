API Documentation

API to identify Fraudulent Transactions

Endpoint:/predict

Method: POST

Request in the following data Format


fiscalYear=2017
fiscalMonth=1
divName=DivisionName
merchant=MerchantName
catDesc=CategoryDescription
transDate=YYYY-MM-DD
amount=123.45

Example cURL Command


curl -X POST http://localhost:5000/predict \
    -F "fiscalYear=2017" \
    -F "fiscalMonth=1" \
    -F "divName=DivisionName" \
    -F "merchant=MerchantName" \
    -F "catDesc=CategoryDescription" \
    -F "transDate=YYYY-MM-DD" \
    -F "amount=123.45"

Example JSON response with the prediction result


{
    "result": "This transaction is approved."
}
